Dán URL Apps Script của Google Sheet để nhận kết quả bài thi tự động.

📋 Hướng dẫn lấy URL (click để mở)
  1. Mở Google Sheet → Extensions → Apps Script
  2. Dán script doPost vào, nhấn Deploy → New deployment
  3. Chọn Execute as: Me và Who has access: Anyone
  4. Nhấn Deploy → copy URL dạng …/exec
